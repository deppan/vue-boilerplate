## Usage
```
vue create --preset deppan/vue-boilerplate [projectName]
```

## Project setup
```
yarn install
```
or
```
npm install
```

### Compiles and hot-reloads for development
```
yarn run serve
```
or
```
npm run serve
```

### Compiles and minifies for production
```
yarn run build
```
or
```
npm run build
```

### Lints and fixes files
```
yarn run lint
```
or
```
npm run lint
```
